# v-focus 自动获取焦点

让输入框插入时自动获取焦点。

该指令用于原生的`<input>`和`<textarea>`元素。

业务中请直接使用`<u-input>`和`<u-textarea>`组件的`autofocus`属性。

## 示例

``` html
输入框：<input v-focus>
```

## API

### Usage

| Params | Type | Default | Description |
| ----- | ---- | ------- | ----------- |
| value | Boolean | `true` | 是否获取焦点 |
