### 编辑模式

``` html
<u-flowable-link ></u-flowable-link>
```

### 只读模式

``` html
<u-flowable-link mode="readonly" value="普通文本"></u-flowable-link>
```

### 变量支持-只读模式

``` html
<u-flowable-link mode="readonly" :value="'支持变量'"></u-flowable-link>
```

### 变量支持-编辑模式

``` html
<u-flowable-link mode="edit" :value="'支持变量'"></u-flowable-link>
```
